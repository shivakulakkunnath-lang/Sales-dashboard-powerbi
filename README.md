# Sales Dashboard Power BI Project

## Project Overview
This Power BI dashboard analyzes sales performance, customer trends, and business insights using interactive visualizations.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features
- Sales KPI tracking
- Revenue analysis
- Product-wise sales performance
- Regional sales analysis
- Interactive filters and slicers

## Key Insights
- Identified top-performing products
- Compared regional sales trends
- Analyzed customer purchasing behavior
